Version 0.16 6th Nov 2016

Snails!

  • New plugin: Automoderator(beta), this plugin lets you define certain rules and auto punishment for breaking those rules
  • New plugin: AutoRole(beta), this plugin can assign people a role on joins, and also using the command (if they already had the role it will get removed): role <role>
  • New Plugin: Logs(beta), use logs (number of messages) to save the n last messages (similar to how hastebin worked but locally hosted on yagpdb instead)
  • Added HTTPS via lets encrypt!
  • Updated moderation plugin, You can now specify a mute role, and give people the role with the commands mute and unmute
  • More tweaks to the website!
  • Fixed a bug where streamer plugin would fail to remove streaming status when the status has them not playing any game

Next i will extend custom commands a little bit by allowing them to execute commands (in either bot context or sender's context) and have access to arguments, there will also be a basic sound board!

Information

YAGPDB Is still in a very early alpha version. The feature list is very small at the moment but that will change with time!

If you have any bugs or suggestions to report please join my discord server here and report them to me (jonas747#3124)


About the permissons: If you don't feel like trusting the bot you can just untick the permissions you don't want to give it. You will not be able to use the moderation features of the bot though unless you give it the permissions it needs of course
